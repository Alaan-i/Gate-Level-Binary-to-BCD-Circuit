# Binary to Decimal using IC and 7-Segment display
Combinational circuit for converting 4-bit binary to decimal. Designed and soldered to perf-board.

Example operation:

0100<sub>2</sub> ➔ 4<sub>10</sub>           |  1001<sub>2</sub> ➔ 9<sub>10</sub>
